# 🌐 DOS Misinformation Hub

**The Misinformation Crisis: How False Information Shapes Digital Society**

The DOS (Digital Objects and Society) Misinformation Hub is an interactive academic dashboard designed to educate users on the mechanics, psychology, and visual indicators of digital misinformation. Built with Streamlit, this platform transforms theoretical research into hands-on learning modules.

---

## 🚀 Interactive Modules

* **📚 Main Page (Crisis Overview):** A digital flashcard dashboard outlining the four core pillars of misinformation: Mechanics of Spread, The Human Element, Visual Misinformation, and Fighting Back.
* **🕵️ Deepfake Detective:** An interactive audit tool that challenges users to identify common AI generative weaknesses, such as anatomical physics (hands), strict physics (corneal reflections), and typographic gibberish.
* **✂️ Context Slider (Cheapfakes):** A hands-on demonstration of the "Realism Heuristic." Users drag a slider to reveal how intentional cropping and forced perspective of *real* photos can fabricate entirely false narratives.
* **🧟 Spot the Troll:** A social media simulation mini-game where users must identify malicious actors (Outrage Bots, Scammers, and Fake Experts) hiding among normal accounts.
* **🔍 Fact-Checking API:** A live integration with the Google Fact Check Tools API, allowing users to search real-world claims and view official debunks.
* **🕸️ Cascade Analysis:** A data visualization module using NetworkX to map the decentralized, viral spread of false information across peer networks.

## 🛠️ Tech Stack

* **Frontend/Framework:** [Streamlit](https://streamlit.io/)
* **Data Processing:** Pandas, SciPy
* **Data Visualization:** Matplotlib, NetworkX
* **APIs & Datasets:** Google Fact Check Tools API, Kaggle API
* **Language:** Python 3
